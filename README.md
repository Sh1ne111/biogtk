# biogtk
A genome toolkit

## Installation
```
git clone https://github.com/Sh1ne111/biogtk.git

cd KMERIA
chmod +x /path/program/biogtk
```

```
#--------------------------------------------------------------------------------------#
Program: biogtk  [ A bio-genome toolkit ]

Version: v0.0.1

Author: Chen Shuai (chensss1209@gmail.com)

Usage:   biogtk <command> [options]

Command:
        genostat        Summary of FASTA/FASTQ.

        glen            Stat FASTA/FASTQ length.

        subseq          Extract sequences from seqID.

        c2p             Convert CDS sequences to PEP sequences.

       #------------------------------------------------------#
       # Please visit the GitHub repository for more details: #
       #       https://github.com/Sh1ne111/biogtk             #
       #------------------------------------------------------#
#----------------------------------------------------------------------------------------#

```
